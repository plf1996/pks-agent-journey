## 系统简介

个人知识管理系统（Personal Knowledge System, PKS）是一个帮助你收集、整理和关联碎片化知识的Web应用。

### 核心功能
- **知识卡片**：创建笔记、保存链接、记录代码
- **双向链接**：建立卡片之间的关联，形成知识网络
- **智能搜索**：快速找到你需要的内容
- **看板管理**：可视化整理你的知识
- **标签系统**：用标签分类和筛选

## 快速开始

### 首次使用

1. **访问系统**
   - 打开浏览器，访问系统地址（默认：`http://localhost:5173`）

2. **注册账号**
   - 点击右上角的"注册"按钮
   - 填写用户名、邮箱和密码
   - 点击"注册"完成账号创建

3. **登录系统**
   - 使用注册的用户名和密码登录

### 界面导航

登录后，你会看到以下导航菜单：

| 菜单项 | 功能描述 |
|--------|----------|
| 仪表盘 | 查看系统统计和最近活动 |
| 卡片管理 | 查看、创建、编辑知识卡片 |
| 看板视图 | 拖拽式管理卡片分类 |
| 标签管理 | 管理标签和标签分类 |
| 全局搜索 | 搜索卡片和标签 |
| 设置 | 个人设置和偏好 |

## 功能指南

### 1. 卡片管理

#### 创建卡片

1. 点击左侧菜单的"卡片管理"
2. 点击右上角的"新建卡片"按钮
3. 填写卡片信息：
   - **标题**：卡片的简短描述
   - **内容类型**：选择笔记/链接/图片/代码
   - **内容**：根据类型填写相应内容
   - **标签**：选择或创建标签
   - **关联卡片**：选择要关联的其他卡片
4. 点击"保存"完成创建

#### 卡片类型说明

| 类型 | 说明 | 示例 |
|------|------|------|
| 笔记 | 普通文本笔记，支持Markdown | 读书笔记、会议记录 |
| 链接 | 保存网页链接 | 文章收藏、资源链接 |
| 图片 | 图片URL或描述 | 设计灵感、截图说明 |
| 代码 | 代码片段，支持语法高亮 | 函数实现、配置示例 |

#### 编辑和删除

- **编辑**：在卡片列表中点击"编辑"图标，或在卡片详情页点击"编辑"
- **删除**：点击卡片上的"删除"图标，确认后删除
- **批量删除**：勾选多个卡片，点击顶部的"批量删除"

### 2. 双向链接

双向链接是PKS的核心功能，帮助你建立知识之间的联系。

#### 创建链接

1. 编辑一张卡片
2. 在"关联卡片"区域，点击"添加链接"
3. 搜索并选择要关联的卡片
4. 保存卡片

#### 查看链接

在卡片详情页，你可以看到：
- **正向链接**：这张卡片引用了哪些卡片
- **反向链接**：哪些卡片引用了这张卡片

点击链接可以快速跳转到关联卡片。

### 3. 标签系统

#### 创建标签

1. 点击左侧菜单的"标签管理"
2. 点击"新建标签"
3. 填写标签信息：
   - **名称**：标签名称
   - **颜色**：选择标签颜色
   - **父标签**：选择父标签创建层级结构

#### 使用标签

- 在创建/编辑卡片时，从标签列表中选择标签
- 可以搜索已有标签或创建新标签
- 点击标签可以筛选包含该标签的所有卡片

### 4. 全局搜索

#### 搜索内容

1. 点击顶部搜索框
2. 输入关键词
3. 选择搜索范围：
   - 全部：搜索卡片和标签
   - 卡片：只搜索卡片
   - 标签：只搜索标签
4. 按回车或点击搜索图标

#### 搜索结果

搜索结果会显示：
- 匹配的卡片（标题和内容）
- 匹配的标签
- 关键词高亮显示

### 5. 看板视图

看板视图让你像使用Trello一样管理卡片。

#### 使用看板

1. 点击左侧菜单的"看板视图"
2. 你会看到多个列表（列）
3. 拖拽卡片在不同列之间移动

#### 自定义列

1. 点击列标题右侧的"设置"图标
2. 可以编辑列名或删除列
3. 点击右侧的"添加列"创建新列

#### 位置保存

拖拽后，卡片位置会自动保存。刷新页面后位置保持不变。

### 6. 数据导出

#### 导出格式

系统支持导出为：
- **JSON**：完整的数据备份，包含所有字段
- **Markdown**：适合文档归档

#### 导出操作

1. 点击左侧菜单的"设置"
2. 选择"数据导出"
3. 选择导出格式
4. 点击"导出"按钮
5. 文件会自动下载

## 常见问题

### Q：忘记密码怎么办？
A：目前版本请联系管理员重置密码。未来版本将支持自助找回密码。

### Q：如何备份我的数据？
A：在"设置"页面使用"数据导出"功能，定期导出JSON格式的完整数据。

### Q：卡片数量有限制吗？
A：没有硬性限制。但建议定期整理，保持系统高效运行。

### Q：支持多人协作吗？
A：当前版本为单用户版本。多人协作功能正在规划中。

### Q：可以离线使用吗？
A：需要联网使用。未来版本将支持离线缓存。

### Q：如何删除账号？
A：请联系管理员处理账号删除请求。

### Q：支持移动端吗？
A：系统采用响应式设计，支持手机和平板访问。
